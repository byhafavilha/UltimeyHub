# 🌐 Rede Sulnet — Conexão de Elite
> Status: Em Desenvolvimento | Versão: 2.0 (Powered by HEKSEL Ecosystem)

Este é o novo portal oficial da **Rede Sulnet**, redesenhado para oferecer uma experiência de luxo e ultra-velocidade. Abandonamos o visual comum para entregar uma interface futurista baseada em **Glassmorphism** e **Dark Mode Premium**.

## 💎 Diferenciais do Projeto
- **Logo Monogramática:** Fusão das letras R e S em metal cromado.
- **Plataforma Neo-Speed:** Cards de planos com hierarquia visual (Silver, Gold e Diamond).
- **Interface Futurista:** Foco em conversão e experiência do usuário (UX) de alto nível.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Tailwind & Custom Properties)
- JavaScript (Efeitos de Scan e Interatividade)
- GitHub Pages para Hospedagem Global
